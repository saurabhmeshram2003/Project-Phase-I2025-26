Frequently Asked Questions

How do I report a bug in the system? +

To report a bug, navigate to the "Bugs" section in the main menu and click on "Report New Bug". Fill in the required details including title, description, severity level, and category. Once submitted, the bug will be assigned to the appropriate team member.

Can I track the status of my reported bugs? +

Yes, you can track all your reported bugs from the dashboard. Each bug shows its current status (Open, In Progress, Resolved, Closed) and you can set up notifications to be alerted when status changes occur.

What user roles are available in the system? +

The system supports three main roles: Tester (can report bugs and add comments), Developer (can be assigned bugs and update their status), and Administrator (has full system access including user management and workflow customization).

Is there a mobile app for the Bug Tracking System? +

Currently, we offer a responsive web application that works seamlessly on mobile browsers. A dedicated mobile app is in development and will be released in the next quarter.
